Requirements
In order to start using Lollipop-PHP you need the following things:

  1. PHP >= 5.4
  2. MySQLi Extension
  3. and your mind!
Getting Started
In your project folder grab a copy of lollipop, you can place it on vendor/lollipop-php NOTE: Lollipop Routing system requires .htaccess recommended content of .htaccess is shown below. 
                    
# Enable running of scripts
AddHandler cgi-script .pl .cgi
Options +ExecCGI +FollowSymLinks

# Disable Indexing of Directories
Options -Indexes

# Enable RewriteEngine
RewriteEngine on

# Enable page access 
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d

# Redirect all pages to index
RewriteRule ^(.*)$/? index.php [L]
Your first application
On your DOCUMENT_ROOT create your index.php file this will be the front of your application 
                    
// Import lollipop
require_once('./vendor/lollipop-php/autoload.php');

// @var array Configuration file
$config = array(
            'dev_tools' => true
        );
        
use \Lollipop\App;
use \Lollipop\Route;

// Initialize application
App::init($config);

// Index page
Route::get('/', function() {
    echo 'Welcome!';
});
