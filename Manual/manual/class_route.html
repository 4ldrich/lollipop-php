\Lollipop\Route Library

Route library offers URL routing and page caching build for tabilty and speed to tune-up your application.


How-to Start?
In order to use \Lollipop\Route service you need first to configure a .htaccess or Apache Configuration File for your application. Below shows recommended .htacces file. 
                    
# Enable running of scripts
AddHandler cgi-script .pl .cgi
Options +ExecCGI +FollowSymLinks

# Disable Indexing of Directories
Options -Indexes

# Enable RewriteEngine
RewriteEngine on

# Enable page access 
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d

# Redirect all pages to index
RewriteRule ^(.*)$/? index.php [L]
This configuration redirects all request to your index.php
