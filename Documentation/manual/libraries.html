\Lollipop Libraries

Lollipop is built with cool collection of libraries


Libraries
  • App - Lollipop web application library
  • Benchmark - A cool way to benchmark things up!
  • Cache - Re-use things and speed up your application
  • Config - Manage configuration
  • Cookie - Manage and eat those cookies
  • Database - Easy way to manage your data through MySQL
  • File - Create and update file
  • FileSystem - Manage file systems
  • Filter - Filter strings
  • HTML - Create DOM elements
  • Inflector - Organize strings
  • Log - Log important messages
  • Number - Format number
  • Page - Render view file
  • Request - Get data passed to your application
  • Route - Routing and page caching
  • Scraper - Scrap DOM elements from other URI
  • Session - A secure way to manage your session
  • Text - Crypt does strings
  • TimeDate - Be on Time!
  • Url - URL or URI?
  • User - Track your users in
